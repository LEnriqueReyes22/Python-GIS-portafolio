# Proyecto 01 – Buffer e Intersección

## Objetivo

Generar un buffer de 2000 metros alrededor de zonas de riesgo de inundación y calcular su intersección con otras capas geográficas utilizando GeoPandas.

## Herramientas utilizadas

- Python
- GeoPandas
- Matplotlib
- Shapely

## Proceso

1. Carga de capas vectoriales.
2. Reproyección a coordenadas métricas.
3. Creación de buffer de 2000 m.
4. Cálculo de intersección espacial.
5. Exportación de resultados.
6. Generación de mapa temático.
