# Python GIS Portafolio
Portafolio de proyectos de automatización geoespacial desarrollados con:

- Python
- GeoPandas
- Rasterio
- QGIS
- Google Earth Engine

## Proyectos

### 1. Buffer e Intersección

Objetivo:
Crear buffers de 2000 m alrededor de zonas de riesgo y calcular intersecciones espaciales.

Herramientas:
- GeoPandas
- Shapely

Resultados:
- Buffer generado
- Intersección espacial
- Exportación a Shapefile

---

### 2. Raster Clip

Objetivo:
Recortar imágenes raster usando límites vectoriales.

Herramientas:
- Rasterio
- GeoPandas

---

### 3. NDVI con Google Earth Engine

Objetivo:
Calcular índices de vegetación para monitoreo ambiental.

Herramientas:
- Google Earth Engine
- Python

---

### 4. Clasificación de Cobertura de Suelo

Objetivo:
Clasificar coberturas terrestres usando imágenes satelitales.

Herramientas:
- Earth Engine
- Python
